# Commands

Custom Claude Code commands for automation and productivity.

## What are Commands?

Commands are executable scripts that extend Claude Code's functionality. They can be:
- Shell scripts
- Node.js scripts
- Python scripts
- Any executable

## Creating a Command

1. Create your script in this directory
2. Make it executable (if needed)
3. Test thoroughly
4. Document usage and parameters

## Best Practices

- Keep commands focused on a single task
- Include error handling
- Provide clear output/feedback
- Document all parameters and options
