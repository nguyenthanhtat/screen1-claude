# screen1-claude

Development repository for Claude Code customizations - skills, commands, and tests.

## Purpose

This repository serves as a development and testing ground for Claude Code plugins before deploying them to the main project repository for team-wide use.

## Structure

```
screen1-claude/
├── skills/          # Custom Claude Code skills
├── commands/        # Custom Claude Code commands
├── tests/           # Tests for skills and commands
└── docs/            # Documentation for custom tools
```

## Workflow

1. Develop and test skills/commands in this repository
2. Validate functionality and test coverage
3. Export working plugins to main project repository
4. Share with team for broader use

## Getting Started

- Use this repository to experiment with Claude Code customizations
- Test thoroughly before deploying to production
- Document each skill/command with clear usage examples
